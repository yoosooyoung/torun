현대작물

- 식물영양소 함량이 낮다.
- 미네랄 함량이 낮다.

통조림, 건조법

- 추출과정은 저온

유기물
- 탄소가 분자구조의 일부를 구성하는 모든 물질
- 유해물질은 인간의 감각으로 쉽게 느낄 수 없다.

채식주의자
- 인간은 먹이사슬의 맨위
- 인간은 잡식성 , 아미노산 + 비타민 => 건강 채식주의자는 그러지 않음
- 음식에 포함되어 있는 영양소는 함유량이 낮다 -> 건강식품 섭취
- 상업적으로 사육되는 가축 -> 항생제와 호르몬
- 유기농 동물 -> 지방중량이 적음
- 지방조직 -> 유해물질 저장

유기농 육류
- 저자방, 필수지방산, 내성이 강한 대장균 박테리아, 육질
- 오메가-3 지방산의 함류량 2~6배
- 오메가-3 : 혈압, 부장맥, 심장마비, 우울증, 조울증, 주의력결핍, 과잉행동장애
            암의 성장, 전이 속도 감소, 수술회복 빠름
- 오메가-3 보조식품 : 다이옥신과 폴리염화비폐닐이 제거되고 오메가 지방산이 보존되어 있는 것
